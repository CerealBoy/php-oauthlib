
#OAuthLib

##Overview

A library to use the PECL OAuth library for dealing with major OAuth providers.

##Requirements

* **php** >= 5.3.2 (tested)
* **oauth** (pecl) >= 1.2.2

##Usage

Please see the example.php file for a walkthrough.
